Recipe 01

Photoshop workstation

Use for: creative tools, editing suites, modular dashboards, maker software.

Palette
monitor grey, teal, pale blue, tool-shadow black.
Layout
floating palettes, image canvas, status bars, tool columns.
Motion
select, mask, drag, composite, undo.

Risk: late-90s skeuomorphism.
Accuracy: keep early desktop modesty visible.

 Recipe 02

Rave flyer pressure

Use for: nightlife identities, event systems, youth brands, campaign pages.

Palette
acid yellow, black, hot orange, UV blue.
Layout
collisions, circles, arrows, stamps, repeated dates.
Motion
pulse, strobe, blink, repeat, scatter.

Risk: generic smiley nostalgia.
Accuracy: keep the cheap local-flyer logic.

 Recipe 03

Soft office GUI

Use for: productivity products, archival tools, admin systems.

Palette
beige, warm white, slate, muted teal.
Layout
tiled windows, control panels, icons, document stacks.
Motion
open, minimize, select, save, route.

Risk: drifting into 1995 office software.
Accuracy: stay chunkier and more transitional.

 Recipe 04

Deconstructed retail calm

Use for: fashion brands, gallery shops, quiet luxury systems.

Palette
white, stone, graphite, unbleached cotton.
Layout
modular shelving, blank margins, tag systems, asymmetry.
Motion
drift, reveal, slow pan, material close-up.

Risk: generic contemporary minimalism.
Accuracy: keep stronger material evidence.

 Recipe 05

Dream lodge interface

Use for: cultural products, fiction-led brands, media archives.

Palette
curtain red, coffee brown, fir green, cream, chrome.
Layout
staged rooms, clue clusters, recurring symbols.
Motion
dissolve, loop, hum, curtain pull, static drift.

Risk: quirky nostalgia.
Accuracy: keep the tension between comfort and unease.

Short prompts pulled from the new corpus.

Design this through a 1990 lens: Photoshop has made images editable, Windows 3.0 has normalized GUI work, the early web has reframed information as linked space, and rave graphics plus deconstruction fashion are pulling culture in opposite but equally systemic directions.
Give me three 1990-informed directions:
1. Photoshop workstation
2. Rave flyer pressure
3. Deconstructed retail calm
For each, explain the typography, material choices, motion behavior, and what would make it feel too late or too nostalgic.
Critique this interface as if it launched in 1990. Is it closer to a Windows 3.0 desktop, a rave information system, a Muji retail field, or a dreamlike broadcast environment?

1990 is pulled between clean systems and dirty circulation.

Desktop visual literacy: overlapping windows, icons, toolbars, palettes, files, drag-and-drop logic

02

Software for image construction: the screen starts behaving like a compositing table

Early network imagination: linked documents, browsers, servers, and distributed information

Rave logistics graphics: phone numbers, maps, all-caps urgency, fluorescent paper, copied repetition

Minimal luxury reset: fashion and interiors cool down into sharper, quieter authority

Example recipes

Generated from the current Flashback design recipes in the 1990 corpus.

Recipe 01

Windows 3.0 productivity suite

Use for: knowledge tools, file-heavy products, dashboards, office AI, admin systems.

Palette
beige, cloud white, slate blue, charcoal.
Type
utility sans, menu labels, dialog copy.
Layout
overlapping windows, tiled panes, visible controls, boxed modules.
Imagery
folders, disks, toolbars, document stacks, pointers.
Motion
open, minimize, drag, copy, confirm.

Risk: sliding into generic retro computing.
Accuracy: late-beige office seriousness and pre-internet desktop density.

 Recipe 02

Photoshop image lab

Use for: creative tools, media editors, asset libraries, imaging products.

Palette
neutral grey, white, black, one electric accent.
Type
compact UI labels, inspector text, tool names.
Layout
canvas plus palettes, rulers, tool strips, crop boundaries.
Imagery
scans, selections, pixel edges, loupe views, composite fragments.
Motion
select, crop, nudge, adjust, preview.

Risk: looking like later polished Adobe suites.
Accuracy: early software bluntness and visible experimentation.

 Recipe 03

Early web information space

Use for: archives, knowledge graphs, research tools, linked notes, documentation products.

Palette
white, black, hypertext blue, muted grey.
Type
plain serif or early sans, clear hierarchy, no decoration burden.
Layout
linked documents, index pages, nested references, lightweight navigation.
Imagery
terminals, linked diagrams, server rooms, document trees.
Motion
jump, load, follow link, branch, return.

Risk: importing late-90s browser chrome.
Accuracy: research-lab plainness and information-first intent.

 Recipe 04

Rave logistics flyer

Use for: event tools, nightlife brands, drop systems, geofenced launches, underground campaigns.

Palette
copier black, fluorescent yellow, acid green, emergency orange.
Type
all-caps sans, stretched text, repeated phone numbers, map labels.
Layout
stacked notices, directions, arrows, meeting points, overprinted symbols.
Imagery
smileys, hazard marks, maps, warehouse silhouettes, repeated stamps.
Motion
flash, pulse, duplicate, distort, scatter.

Risk: becoming generic acid-house nostalgia.
Accuracy: route logic, urgency, and bad reproduction.

 Recipe 05

Boutique minimal showroom

Use for: fashion commerce, premium retail, cultural brands, portfolios, gallery tools.

Palette
white, black, stone, denim, muted metal.
Type
restrained sans or serif with generous spacing.
Layout
empty margins, sharp crops, object isolation, strong baseline discipline.
Imagery
garment rails, concrete floors, raw walls, cast shadows, careful staging.
Motion
slow reveal, slide, fade, pause.

Risk: feeling like generic luxury minimalism.
Accuracy: anti-glamour attitude and 1990's cooler editorial severity.
