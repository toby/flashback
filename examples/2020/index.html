Example recipes

Direct entry points into the 2020 corpus recipes.

Recipe 01

Video tile workspace

Use for: collaboration tools, event systems, education products.

Palette
white, navy, muted blue, alert amber.
Layout
tiles, chat panels, agenda strips, modal overlays.
Motion
join, mute, pin, share, react.

Risk: generic enterprise UI.
Accuracy: keep the domestic context visible.

 Recipe 02

Domestic calm dashboard

Use for: wellness products, home tools, planning systems.

Palette
beige, moss, fog grey, soft blue.
Layout
roomy cards, gentle charts, notebook spacing.
Motion
check in, log, expand, reflect, schedule.

Risk: generic self-care minimalism.
Accuracy: include contingency language.

 Recipe 03

TikTok kinetic layer

Use for: creator tools, youth campaigns, music marketing.

Palette
black, white, creator cyan, signal pink, caption yellow.
Layout
vertical frame, stacked captions, timing beats.
Motion
jump cut, caption pop, swipe, duet, loop.

Risk: platform tropes without timing intelligence.
Accuracy: keep creator immediacy.

 Recipe 04

Civic care system

Use for: public-service products, health tools, explainers.

Palette
white, civic navy, chart red, warning amber.
Layout
charts, maps, update modules, FAQ rails.
Motion
update, expand, filter, alert, confirm.

Risk: sterile data theater.
Accuracy: keep human guidance visible.

 Recipe 05

Contactless threshold

Use for: retail, hospitality, mobility, pickup and delivery systems.

Palette
black, white, kraft, steel grey, safety green.
Layout
single-purpose steps, pickup states, code blocks.
Motion
scan, confirm, arrive, collect, leave.

Risk: generic fintech UX.
Accuracy: emphasize threshold signage logic.

Prompt seeds

Three quick 2020-oriented prompts.

Design this through a 2020 lens: screens have become the room for work, school, care, and performance; video tiles and contactless systems organize daily life; and short-form vertical motion has reset the pace of visual culture.
Give me three 2020-informed directions:
1. Video tile workspace
2. Domestic calm dashboard
3. Civic care system
For each, define the tone, layout, motion language, and what would make it feel too generic or too late-decade.
Critique this interface as if it launched in 2020. Is it closer to a remote-work room, a contactless threshold, a creator-video layer, or a civic explainer?

Anti-cliches

Keep the year concrete and behaviorally specific.

  • Generic pandemic-core sadness
  • Ultra-polished creator branding with no domestic texture
  • Dashboards that hide uncertainty
  • TikTok references reduced to random stickers
  • Remote-work visuals without tile logic or mute states
